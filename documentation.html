User Guide

Guide Utilisateur

Overview

Easy Task is your personal assistant for managing deadlines, meetings, and invoices. It helps you stay organized with visual countdowns and professional tools.

Key Features

  • Deadline Management: Visual countdowns, progress gauges, and overlap protection to avoid double-booking.
  • Invoicing: Create professional invoices, manage items with different prices, and share them via Email or PDF.
  • Focus Mode: A dedicated timer view to help you concentrate on the task at hand.
  • Google Integration: Sync reminders with your Google Calendar.

Vue d'ensemble

Easy Task est votre assistant personnel pour gérer vos échéances, réunions et factures. Il vous aide à rester organisé grâce à des comptes à rebours visuels et des outils professionnels.

Fonctionnalités Clés

  • Gestion des Échéances : Comptes à rebours visuels, jauges de progression et protection contre les chevauchements.
  • Facturation : Créez des factures professionnelles, gérez des articles avec des prix différents, et partagez-les par Email ou PDF.
  • Mode Concentration : Une vue minuteur dédiée pour vous aider à vous concentrer sur la tâche en cours.
  • Intégration Google : Synchronisez vos rappels avec votre Google Agenda.

Technical Documentation

Documentation Technique

Architecture (MVVM)

The app follows the Model-View-ViewModel pattern using SwiftUI.

  • Models: Define data structures (Deadline, Invoice).
  • Views: Display the UI and observe Managers (HomeView, InvoiceDetailView).
  • Managers (ViewModels): Handle business logic and data persistence (DeadlineManager, InvoiceManager).

Architecture (MVVM)

L'application suit le modèle Model-View-ViewModel avec SwiftUI.

  • Modèles : Définissent les structures de données (Deadline, Invoice).
  • Vues : Affichent l'interface et observent les Managers (HomeView, InvoiceDetailView).
  • Managers (ViewModels) : Gèrent la logique métier et la persistance (DeadlineManager, InvoiceManager).

Data Persistence

Persistance des Données

Local (UserDefaults) Used for Invoices and Deadlines. Simple JSON encoding. Utilisé pour les Factures et les Échéances. Encodage JSON simple.

Cloud (Firebase) Used for User Profiles and Premium Subscription status. Utilisé pour le Profil Utilisateur et le statut d'Abonnement Premium.

Example: Invoice Persistence (Local)

Exemple : Persistance Facture (Local)

private func saveInvoices() {
    invoices.sort { $0.createdAt > $1.createdAt }
    if let encoded = try? JSONEncoder().encode(invoices) {
        UserDefaults.standard.set(encoded, forKey: userDefaultsKey)
    }
}

Overlap Prevention (DeadlineManager)

Prévention des Chevauchements (DeadlineManager)

Prevents users from scheduling two tasks at the same time.

Empêche les utilisateurs de planifier deux tâches en même temps.

func checkForOverlap(startDate: Date, duration: TimeInterval) -> Bool {
    let newEnd = startDate.addingTimeInterval(duration)
    for deadline in deadlines {
        // Checking intersection: (StartA < EndB) && (EndA > StartB)
        let taskEnd = deadline.date.addingTimeInterval(deadline.duration)
        if startDate < taskEnd && newEnd > deadline.date {
            return true
        }
    }
    return false
}
